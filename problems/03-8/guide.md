## 考え方

この問題では、4 桁の整数の各桁を下の位から順に取り出します。  
`% 10` を使うと、その数の一番右の 1 桁を取り出せます。

たとえば、一の位は `n % 10` です。  
十の位は `n / 10` で 1 桁右へずらしてから `% 10` を使うので、`n / 10 % 10` と書けます。

同じ考え方で、

- 一の位: `n % 10`
- 十の位: `n / 10 % 10`
- 百の位: `n / 100 % 10`
- 千の位: `n / 1000 % 10`

と取り出せます。

## つまずきやすい点

- `/` は整数どうしなら整数の割り算になります
- `% 10` は「最後の 1 桁を取り出す」と考えると分かりやすいです
- 出力順は千の位からではなく、一の位からです
- 0 を含む桁もそのまま出力します

## 模範解答

```c
#include <stdio.h>

int main()
{
	int n;
	scanf("%d", &n);
	printf("%d\n", (n % 10));
	printf("%d\n", (n / 10 % 10));
	printf("%d\n", (n / 100 % 10));
	printf("%d\n", (n / 1000 % 10));
}
```
