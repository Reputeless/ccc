## 考え方

この問題では、入力された整数を 1 つ読み取り、その値を 2 倍して出力します。  
まず `scanf` で `x` を読み込み、そのあと `x * 2` を `printf` で表示します。

## つまずきやすい点

- `scanf` で 1 つの整数を読むときも、変数には `&` を付けます
- 2 倍は `x + x` でも表せますが、ここでは `x * 2` と書くと分かりやすいです
- 出力の最後に `\n` を入れておきましょう

## 模範解答

```c
#include <stdio.h>

int main()
{
	int x;
	scanf("%d", &x);
	printf("%d\n", (x * 2));
}
```

## 別の書き方

式のまわりの `()` は、付けなくても正しく動きます。

```c
#include <stdio.h>

int main()
{
	int x;
	scanf("%d", &x);
	printf("%d\n", x * 2);
}
```
