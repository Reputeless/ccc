## 考え方

長方形の面積は、縦と横を掛け算すると求められます。  
この問題では、`scanf` で `w` と `h` を読み込み、そのあと `w * h` を `printf` で出力すれば十分です。

## つまずきやすい点

- `scanf` の変数には `&` を付けます
- 面積なので、足し算ではなく掛け算 `*` を使います
- 出力の最後に `\n` を入れておきましょう

## 模範解答

```c
#include <stdio.h>

int main()
{
	int w, h;
	scanf("%d %d", &w, &h);
	printf("%d\n", (w * h));
}
```
