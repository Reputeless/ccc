## 考え方

この問題では、入力された 2 つの整数をそのまま逆の順で表示します。  
まず `scanf` で `a` と `b` を読み込み、そのあと `printf` では `b`, `a` の順に並べます。

計算は必要ありません。  
読み込んだ 2 つの値を、そのまま並べ替えて出力すればよいです。

## つまずきやすい点

- `scanf` で読む順番は `a`, `b` です
- 出力するときの順番は `b`, `a` に変わります
- 2 つの整数の間に半角空白を入れます

## 模範解答

```c
#include <stdio.h>

int main()
{
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d %d\n", b, a);
}
```
