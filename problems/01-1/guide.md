## 考え方

この問題では、文字列 `Apple` をそのまま 1 行で表示できれば十分です。  
入力を読んだり、計算したりする必要はありません。

`puts` を使うと、文字列を表示して最後に改行も付けてくれるので、この問題に向いています。

## つまずきやすい点

- `puts("Apple");` のように、表示したい文字列は `"` で囲みます
- `puts(Apple);` のように `"` を忘れるとエラーになります
- `puts` は改行付きで表示するので、`Apple` を 1 行出力する今回の条件に合っています

## 模範解答

```c
#include <stdio.h>

int main()
{
	puts("Apple");
}
```
