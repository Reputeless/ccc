## 問題

この問題は、`CCC` で **表** と **画像** がどのように表示されるかを確認するためのサンプルです。

下の図は、横 4 マス・縦 3 マスの格子を表しています。

![格子の図](./figure-grid.svg)

## 表示例

次の表は、格子の横幅と縦幅からマスの個数を求める例です。

| 横 | 縦 | マスの個数 |
| --- | --- | --- |
| 4 | 3 | 12 |
| 5 | 2 | 10 |
| 8 | 1 | 8 |

## 入力

この問題に入力はありません。

## 出力

文字列 `OK` を 1 行で出力してください。

## ヒント

表示確認用の問題なので、まずは次のように固定文字列を出力すれば十分です。

```c
#include <stdio.h>

int main(void) {
	printf("OK\n");
	return 0;
}
```
