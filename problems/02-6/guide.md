## 考え方

この問題では、計算した整数を 2 行に分けて表示します。  
`printf("%d\n", ...);` を使うと、整数を表示して最後に改行できます。

1 行目では `123 + 456` を、2 行目では `123 - 456` を計算して表示します。  
計算式は、あらかじめ紙で計算してから数だけを書くのではなく、そのままプログラムの中に書いてかまいません。

## つまずきやすい点

- `puts` ではなく `printf` を使う問題です
- `%d` は整数用の書式指定子です
- 2 行目の引き算は負の数になりますが、正の数と同じように `%d` で表示できます

## 模範解答

```c
#include <stdio.h>

int main()
{
	printf("%d\n", 123 + 456);
	printf("%d\n", 123 - 456);
}
```

## 別の書き方

暗算できるなら、次のように結果をそのまま文字列で書いても正しく出力できます。

```c
#include <stdio.h>

int main()
{
	puts("579");
	puts("-333");
}
```

ただし、この問題ではコンピュータに計算を任せる書き方を練習する方が自然です。  
数が大きくなったときも、プログラムに計算させる方が正確で安心です。
