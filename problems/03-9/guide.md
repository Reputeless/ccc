## 考え方

この問題では、入力される `s` は「翌日になるまであと何秒か」を表しています。  
1 日は `86400` 秒なので、現在がその日の何秒目かは `86400 - s` で求められます。

たとえば、この値を `t` とすると、

- 時: `t / 3600`
- 分: `t / 60 % 60`
- 秒: `t % 60`

と計算できます。

出力は `hh:mm:ss` の形式なので、時、分、秒をそれぞれ 2 桁でそろえる必要があります。  
そのため、`printf("%02d:%02d:%02d\n", hour, minute, second);` のように `%02d` を使って出力します。

## 模範解答

```c
#include <stdio.h>

int main()
{
	int s;
	scanf("%d", &s);

	int t = (86400 - s);
	int hour = (t / 3600);
	int minute = (t / 60 % 60);
	int second = (t % 60);

	printf("%02d:%02d:%02d\n", hour, minute, second);
}
```
