## 考え方

この問題では、決まった 2 つの文字列を上から順に表示します。  
入力も計算もないので、`Apple` と `Banana` をそのまま出力すれば大丈夫です。

`puts` を使うと、文字列を表示したあと自動で改行が入ります。  
そのため、`puts("Apple");` と `puts("Banana");` を順に書けば、ちょうど 2 行で出力できます。

## つまずきやすい点

- `Apple` と `Banana` は、それぞれ `"` で囲んで書きます
- 1 行でまとめて出力するのではなく、2 行に分けて出力します
- `puts` を 2 回使うと、自然に 1 行目と 2 行目を分けられます

## 模範解答

```c
#include <stdio.h>

int main()
{
	puts("Apple");
	puts("Banana");
}
```
