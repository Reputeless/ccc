## 問題

あるゲームを何日か続けて遊びました。  
各日の結果は、勝ちなら `o`、負けなら `x` で記録されています。

結果を表す文字列 `s` が与えられるので、`o` が連続している区間の長さの最大値を求めてください。

## 制約

- `s` の長さは `1` 以上 `100` 以下
- `s` は `o` と `x` だけからなる

## 入力

入力は次の形式で標準入力から与えられます。

```text
s
```

## 出力

`o` が連続している区間の長さの最大値を出力してください。

## ヒント

左から 1 文字ずつ見て、今何個連続しているかを数えると考えやすいです。  
`x` が来たら連続数を 0 に戻します。

## 考え方

文字列を左から順に見ていき、`o` が続いている間は連続数を増やします。  
`x` が出たら連続は切れるので、連続数を 0 に戻します。

このとき、

- 今の連続数
- これまでの最大値

の 2 つを持っておくと、1 回の走査で答えを求められます。

## つまずきやすい点

- `o` を見たときは連続数を 1 増やします
- `x` を見たときは連続数を 0 に戻します
- 最大値の更新を忘れないようにします

## 模範解答

```python
s = input()

current = 0
best = 0

for ch in s:
    if ch == "o":
        current += 1
        best = max(best, current)
    else:
        current = 0

print(best)
```
